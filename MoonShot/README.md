# Project 8 - Moonshot

## Topics
ScrollView, Navigation, Codable & JSON, Grid, Generics, DateFormatter.

## Description
This is Project 8 from the "100 Days of SwiftUI" challenge (https://www.hackingwithswift.com/100/swiftui/39).

This project is a space exploration application that displays detailed information about NASA's Apollo missions. The goal is to build skills in using ScrollView, NavigationStack, complex layouts with grids, and decoding JSON data using Generics.

The app loads data from local JSON files to show a list of missions. Users can select a mission to view its details, including the launch date, description, and crew members. Tapping on a crew member reveals their specific astronaut details.
